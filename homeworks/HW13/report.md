# HW13 — токенизация, BERT-инференс и fine-tuning для классификации текста

## 1. Кратко: что сделано

- Загружен датасет `emotion` (Hugging Face Datasets).
- Показана токенизация (примеры токенов/`input_ids`/`attention_mask`) для нескольких текстов.
- Проведён инференс предобученной моделью sentiment (SST-2) как демонстрация пайплайна.
- Выполнен fine-tuning `distilbert-base-uncased` под 6-классовую классификацию эмоций, оценка на test.
- Артефакты сохраняются в `./artifacts/`.

## 2. Среда и воспроизводимость

- Seed: `42`
- Устройство: CPU/GPU определяется автоматически (`cuda` при наличии)
- Как запустить: открыть `HW13.ipynb` и выполнить Run All

## 3. Данные

- Датасет: `emotion` (train/validation/test сплиты уже заданы)
- Тексты: короткие пользовательские фразы на английском
- Цель: предсказать эмоцию (6 классов)

## 4. Модель и обучение

- База: `distilbert-base-uncased`
- Токенизация: `AutoTokenizer`
- Обучение: `Trainer` (evaluation по эпохам, best model по accuracy на validation)

## 5. Результаты и артефакты

После запуска ноутбука/скрипта:

- Матрица ошибок: `./artifacts/figures/confusion_matrix.png`
- Примеры предсказаний: `./artifacts/sample_predictions.csv`
- Логи/чекпойнты Trainer: `./artifacts/hf/`

## 6. Итоговый вывод

Fine-tuning небольшой BERT-подобной модели даёт заметный прирост качества по сравнению с использованием “чужого” классификатора (SST-2) на неродной задаче; корректная токенизация и валидация по epoch — базовая основа для стабильного результата.
