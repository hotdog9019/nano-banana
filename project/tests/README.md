# Тесты

Тесты запускаются из корня `project`:

```bash
pytest tests -q
```

Что проверяется:

- `test_data.py` - датасет содержит нужные колонки, 100 строк и все 5 классов; split сохраняет классы в train/test.
- `test_model.py` - модель обучается, метрики имеют корректный диапазон, ответ `predict_ticket` содержит категорию и confidence.
- `conftest.py` - добавляет корень проекта в `sys.path`, чтобы `pytest` стабильно импортировал пакет `src`.
