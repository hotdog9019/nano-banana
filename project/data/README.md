# Данные проекта

В этой папке лежит демонстрационный датасет `support_tickets.csv`.

Файл содержит 100 синтетических обращений в поддержку на русском и английском языках. Колонки:

- `ticket_id` - идентификатор обращения.
- `text` - текст обращения пользователя.
- `category` - целевая категория для классификации.
- `priority` - демонстрационный приоритет обращения.

Данные не являются реальными обращениями пользователей и не содержат персональных или конфиденциальных сведений. Полный внешний датасет для запуска проекта не нужен.
