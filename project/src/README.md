# Код проекта

Основные модули:

- `config.py` - загрузка YAML-конфига и разрешение путей относительно папки `project`.
- `data.py` - чтение CSV, проверка обязательных колонок и стратифицированный train/test split.
- `model.py` - построение baseline, кандидатных моделей, финальной модели, метрик и матрицы ошибок.
- `train.py` - точка входа `python -m src.train`, сохраняет модель и артефакты.
- `predict.py` - загрузка сохраненной модели и единый контракт ответа `/predict`.
- `service.py` - HTTP API на стандартной библиотеке Python.

Команды запуска описаны в `project/README.md`.
