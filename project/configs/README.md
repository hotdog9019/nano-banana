# Конфигурация

- `config.yaml` - основной конфиг проекта: пути к данным и артефактам, параметры split, TF-IDF, классификатора и HTTP-сервиса.
- `.env.example` - пример переменных окружения для запуска сервиса без реальных секретов.

Файл `.env` с локальными значениями не нужно добавлять в репозиторий.
